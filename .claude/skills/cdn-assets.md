# CDN Assets Skill

The `public/cdn-assets/` folder contains files that get deployed to the CDN. It holds two kinds of content with strictly different rules.

## Hard Rules

- **NEVER** place raw 3D models (`.fbx`, `.obj`, `.gltf`, `.glb`), mesh textures, or audio files directly in this folder
- **NEVER** manually edit `.stow` files — they are auto-generated by `npx stowkit build`
- 3D models, mesh textures, and audio **MUST** go through the StowKit pipeline (`assets/` folder → `npx stowkit build` → `.stow` output lands here automatically)
- Only **UI assets** (icons, button images, HUD elements) are placed here manually

## What Belongs Here

| Content | How it gets here | How to load it |
|---|---|---|
| `.stow` packs | Auto-generated by `npx stowkit build` | `loadStowKitPack('default')` → `pack.loadMesh(id)`, etc. |
| UI images (`.png`, `.jpg`) | Manually added by you | `RundotGameAPI.cdn.fetchAsset('filename.png')` |

## Adding a UI Asset

1. Place the image file directly in `public/cdn-assets/`
2. Load it in code:

```typescript
import RundotGameAPI from '@series-inc/rundot-game-sdk/api';

const blob = await RundotGameAPI.cdn.fetchAsset('icon-play.png');
const url = URL.createObjectURL(blob);

// In React
<img src={url} alt="Play" />
```

## Loading from StowKit Packs

`.stow` files in this folder are loaded via `loadStowKitPack` — see the `asset-pipeline` skill for full details.

```typescript
import { loadStowKitPack } from '../loadStowKitPack';

const pack = await loadStowKitPack('default');

const mesh = await pack.loadMesh('my_model');
const texture = await pack.loadTexture('my_texture');
const audio = await pack.loadAudio('sfx_click', listener);
```

## Decision Guide

| The asset is... | Where it goes | Loaded via |
|---|---|---|
| A 3D model (FBX, OBJ, GLTF, GLB) | `assets/` → StowKit pipeline | `pack.loadMesh()` / `pack.loadSkinnedMesh()` |
| A texture for a 3D mesh | `assets/` → StowKit pipeline | Loaded automatically with mesh via `.stowmat` |
| Audio (SFX, music) | `assets/` → StowKit pipeline | `pack.loadAudio()` |
| An animation clip | `assets/` → StowKit pipeline | `pack.loadAnimation()` |
| A UI icon / button image / HUD element | Directly into `public/cdn-assets/` | `RundotGameAPI.cdn.fetchAsset()` |

## Deploying

All files in `public/cdn-assets/` are uploaded to the CDN when you run:

```bash
rundot deploy
```
